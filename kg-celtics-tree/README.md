# Celtics — Kevin Garnett Trade Tree

Standalone visualization of the **Boston Celtics** side of the July 31, 2007 Kevin Garnett trade.

**Patterns & layout from:** [Roster Routes](../app/) — orthogonal edges, node styling, headshots. No shared code; this is self-contained.

**Run:** `npm install && npm run dev` — opens on http://localhost:3001 (Roster Routes stays on 3000)
